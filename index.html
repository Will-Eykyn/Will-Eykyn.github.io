AIS Maritime Tracker

Built a Python data pipeline that ingests live AIS (Automatic Identification System) vessel position data, processes and stores it, and visualizes ship movements in real time. The system handles continuous streaming data and is designed to monitor maritime traffic patterns.

Python Data Pipelines Geospatial APIs Docker

Sail Performance ML Model — North Sails

During an internship at North Sails (Newport, RI), developed machine learning models to predict batten stiffness from physical sail measurements. Used ML.NET with LGBM (Light Gradient Boosted Machine) to train one model per batten position within each sail type, validated results with R² analysis, and built Excel workbooks to track model predictions against actual outputs.

C# ML.NET LGBM SQL Server Machine Learning Data Validation

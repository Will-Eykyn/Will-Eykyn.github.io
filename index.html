Using Machine Learning to Design Sails – North Sails

During an internship at North Sails, developed machine learning models to predict batten stiffness from physical sail measurements. Used ML.NET with LGBM (Light Gradient Boosted Machine) to train models across batten positions, validated results with R² analysis, and built Excel workbooks to track predictions against actual outputs. The solution is now integrated into North Sails' internal design software and used by sail designers globally.

C# ML.NET LGBM SQL Server Machine Learning Data Validation
